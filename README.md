Welcome to the **[ESP32 For Busy People](https://techexplorations.com/so/esp32fbpev/)** course repository! This repository contains all the code and resources you need to get started with ESP32 development, organized for clarity and ease of use.

## Repository Structure

### **ESP32S3 Scripts (made for Core 3.x)**
- **Location:** [`ESP32S3 scripts (made for Core 3.x)`](https://github.com/futureshocked/ESP32-For-Busy-People-1/tree/master/ESP32S3%20scripts%20(made%20for%20Core%203.x))
- **Description:** This folder contains scripts specifically designed for the **ESP32S3** board. These scripts are compatible with **ESP32 Arduino Core 3.x**. Use these if you are working with the ESP32S3 or want to take advantage of the latest features in Core 3.x.

### **Original ESP32 Sketches (made for Core 1.x)**
- **Location:** [`Original ESP32 sketches (made for Core 1.x)`](https://github.com/futureshocked/ESP32-For-Busy-People-1/tree/master/Original%20ESP32%20sketches%20(made%20for%20Core%201.x))
- **Description:** This folder contains the original sketches for the **ESP32** board, compatible with **ESP32 Arduino Core 1.x**. Use these if you are working with the original ESP32 or need to maintain compatibility with Core 1.x.

### **Schematics**
- **Location:** [`Schematics`](https://github.com/futureshocked/ESP32-For-Busy-People-1/tree/master/Schematics)
- **Description:** Here you will find all the schematics referenced in the course, to help you wire up your projects correctly.

## Getting Started
- If you are using an **ESP32S3**, head to the **ESP32S3 scripts** folder.
- If you are using the **original ESP32**, check out the **Original ESP32 sketches** folder.
- For wiring and hardware setup, refer to the **Schematics** folder.
